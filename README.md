## 🌟 Project Overview
Welcome to the **Job Portal**!  

A **modern platform** that connects **job seekers** with **employers**. Users can:  
- Explore jobs  
- Apply easily  
- Manage profiles  
- Employers can post listings & track applications  

**Built with Python, HTML, and CSS** for a smooth, responsive experience.
